1.  [ghost](https://github.com/TryGhost/Ghost)

## setup
```console
$ # global
$ npm install ghost-cli -g
$ # init submodule
$ git submodule update --init --recursive
$ # build
$ npm run setup
```

## config
```console
$ ghost setup
```

## run

[详细文档](https://docs.ghost.org/v1.0.0/docs/ghost-cli)

```console
$ ghost start
```
